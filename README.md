# ProCart
The ProCart Web app plots a cartoon to illustrate the residue properties of amyloid proteins based on their atomic structures.

Click the link (<a href="https://protein-structure-procart.herokuapp.com">ProCart</a>) to plot a cartoon of your amyloid protein structure!</a>
